# DreamDay - Tages-Assistent App

## Projektidee

DreamDay ist eine Web-Applikation zur Tagesplanung, die Wetterdaten, Musik und Rezepte kombiniert, um dem Nutzer einen optimalen Tagesablauf bereitzustellen.

## Die 3 API-Auswahl

### 1. Open-Meteo Weather API
- Kostenlos, keine Authentifizierung erforderlich
- Liefert Wetterdaten in Echtzeit
- Einfache REST-API Integration
- https://open-meteo.com

### 2. Spotify Web API
- Zugriff auf umfangreiche Song-Datenbank
- Playlist-Generierung möglich
- OAuth 2.0 Authentifizierung
- https://developer.spotify.com

### 3. Spoonacular API
- Rezept-Datenbank mit Nährwertinformationen
- Meal-Planning basierend auf Vorlieben
- Free Tier mit 5000 Requests/Monat
- https://spoonacular.com/food-api

## Begründung der Auswahl

Diese drei APIs bilden eine sinnvolle Kombination für eine Tages-Assistent-Anwendung:
- Open-Meteo liefert Kontextinformationen (Wetter)
- Spotify ermöglicht personalisierte Unterhaltung
- Spoonacular unterstützt Mahlzeitplanung

Die Auswahl bevorzugt kostenlose oder großzügig kostentenlose Tiers, um Anforderungen für ein Schulprojekt zu erfüllen.
